<summary>Booting Fedora on Zhihe A210</summary>

This tutorial provides a step-by-step guide on how to flash system images onto the Zhihe A210 development board.

-----

# 0x01 Enter Burning Mode

Before proceeding, please refer to the [A210 SODIMM V2 Burning Instruction](https://developer.zhcomputing.com/docs/A210/A210_Burning_Instruction/A210_SODIMM_V2_Burning_Instruction) to ensure your development board has correctly entered Burning Mode.

-----

# 0x02 Flash Firmware

Execute the following commands to download and flash the required firmware components:

```
# Download firmware files
wget https://mirror.iscas.ac.cn/fedora-riscv/dl/Zhihe-A210/Zhihe-A210/fw/bootzero-rvbl.bin
wget https://mirror.iscas.ac.cn/fedora-riscv/dl/Zhihe-A210/Zhihe-A210/fw/spl-with-fit-rvbl.bin
wget https://mirror.iscas.ac.cn/fedora-riscv/dl/Zhihe-A210/Zhihe-A210/fw/btz-with-uboot-rvbl.bin

# Flash firmware
fastboot flash ram bootzero-rvbl.bin
fastboot reboot
fastboot flash ram spl-with-fit-rvbl.bin
fastboot reboot
sleep 5
fastboot flash mmc0boot0 btz-with-uboot-rvbl.bin
```

-----

# 0x03 Flash System Image

Ensure the device remains in Burning Mode. Download your preferred system image, decompress it, and flash it using the following commands.

**Replace the wildcard (*) in the commands below with the actual filename of your downloaded image.**

```
# Decompress the image
gzip -d Fedora-*.raw.gz

# Flash to MMC
fastboot flash mmc0 Fedora-*.raw
fastboot reboot
```

-----

# 0x04 Post-Flash Configuration

Upon the first boot, you may need to interrupt the U-Boot startup sequence to reset the environment variables. Once you reach the U-Boot prompt, execute:

```
env erase
reset
```
